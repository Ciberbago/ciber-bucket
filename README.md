# Ciberbucket

Personal scoop bucket for apps that I cannot find in the way I want in official buckets.

## Included programs

- Clickpaste
- LdPlayer9
- PMX Editor in english
- RDP Plus
- Rustdesk non portable
- Steam Cleaner
- Tinywall
- Win PS2EXE
- Windhawk

## Installation

```
scoop bucket add ciber https://github.com/Ciberbago/ciber-bucket/
```
